Seletores de CSS

Um seletor de CSS seleciona o(s) elemento(s) HTML que você deseja estilizar.

Categorias

Seletores Simples

Seleciona elementos com base no nome, id, classe...

Seletor de Elementos

Seleciona elementos HTML com base no nome.

Exemplo
p {
text-align: center;
color: red;
}

Seletor de Id

O seletor de id usa o atributo de um elemento em HTML para selecionar um elemento específico.
Para selecionar um Id, escreva um caractere hash (#) seguido do id do elemento.

Exemplo
#para1 {
text-align: center;
color: red;
}

Seletor de Classe

O Seletor de Classe seleciona elementos HTML com um atributo de classe específico.
Para selecionar elementos com uma classe específica, escreve um caractere de ponto (.), seguido do nome da classe.

Exemplo
.center {
text-align: center;
color: red;
}

Pode-se especificar que apenas elementos HTML específicos devem ser afetados por uma classe.

Exemplo
p.center {
text-align: center;
color: red;
}

Os elementos HTML também podem se referir a mais de uma classe.

Exemplo
<p class="center large">This paragraph referes to two classes</p>

Seletor Universal

O Seletor Universal (*) seleciona todos os elementos HTML da página.

Exemplo
* {
text-align: center;
color: red;
}

Seletor de Agrupamento

O Seletor de Agrupamento seleciona todos os elementos HTML com as mesmas definições de estilo.

h1 {
text-align: center;
color: red;
}

Seletores de Combinação

Um seletor CSS pode conter mais de um seletor simples. Entre os seletores simples, pode-se incluir um combinador.

Existem 4 combinadores diferentes em CSS:

  • descendant selector
  • child selector (>)
  • adjacent sibling selector (+)
  • general sibling selector (~)

Descendant Selector

Corresponde a todos os elementos que são descendentes de um elemento especificado.

Exemplo

O exemplo a seguir seleciona todos os elementos <p> dentro dos elementos <div>.

Seleciona elementos com base em um determinado estado.

div p {
background-color: yellow;
}

Child Selector

Seleciona todos os elementos que são filhos de um elemento especificado.

Exemplo

O exemplo a seguir seleciona todos os elementos <p> que são filhos de um elemento <div>.

div > p {
background-color: yellow;
}

Adjacent Sibling Selector

É usado para selecionar um elemento que está diretamente após outro elemento específico.

Devem ter o mesmo elemento "pai".

Exemplo

O exemplo a seguir seleciona o primeiro elemento <p> que é colocado imediatamente após os elementos <div>.

div + p {
background-color: yellow;
}

General Sibling Selector

Seleciona todos os elementos que são os próximos irmãos de um elemento especificado.

Exemplo

O exemplo a seguir seleciona todos os elementos <p> que são os próximos irmãos dos elementos <div>.

div ~ p {
background-color: yellow;
}

Seletores de Pseudoclasse

É usado para definir um estado especial de um elemento.

Por exemplo, pode ser usado para:

  • Estilizar um elemento quando um usuário passa o mouse sobre ele;
  • Estilizar links visitados e não visitados de forma diferente;
  • Estilizar um elemento quando ele recebe o foco.

Sintaxe

selector:pseudo-class {
property: value;
}

Pseudo-Classes Àncora

Exemplos
/* unvisited link */
a:link {
color: #FF0000;
}

/* visited link */
a:visited {
color: #00FF00;
}

/* mouse over link */
a:hover {
color: #FF00FF;
}

/* selected link */
a:active {
color: #0000FF;
}

Seletores de Pseudo-Elementos

Seleciona e estiliza parte de um elemento.

Seletores de Atributo

Seleciona elementos com base em um atributo ou valor de atributo.
